개요

Exchango의 개인정보 보호 정책 및 데이터 처리 방식에 대한 정보입니다

이 개인정보 처리방침은 Exchango 환율 변환 앱 사용 시 개인정보의 수집, 사용, 보호 방식을 명시합니다. 앱 개선을 위해 Google Firebase Analytics 등 분석 도구를 사용하며, 모든 데이터는 익명으로 처리됩니다.

개인정보 보호 우선 설계

Exchango는 개인정보 보호를 우선으로 설계되었습니다. 사용자 데이터는 기기에만 저장되며, 개인식별정보는 외부 서버에 수집, 저장, 전송되지 않습니다.

개인정보 보호 원칙

저희가 유지하는 핵심 개인정보 보호 원칙들입니다

로컬 저장만

모든 데이터는 사용자 기기에만 저장됩니다. 외부 저장소나 클라우드 동기화는 지원하지 않습니다.

익명 분석 데이터만

개인식별정보는 수집하지 않으며, 앱 개선을 위한 익명화된 사용 통계만 수집합니다.

보안 설계

사용자 정보 보호를 위해 보안 기술을 적용해 설계되었습니다.

투명성

데이터 처리 방식을 투명하게 공개하여 신뢰성을 유지합니다.

수집하지 않는 정보

저희가 절대 수집하지 않는 정보들을 명확히 알려드립니다

Exchango는 다음 정보들을 수집하지 않습니다:

개인 식별 정보

이름, 이메일, 전화번호 등 개인을 식별할 수 있는 모든 정보

금융 정보

금융 계좌 정보, 거래 내역, 신용카드 정보

위치 데이터

정확한 위치 데이터, GPS 좌표

기기 식별자

개별 기기 식별자, 광고 ID

수집할 수 있는 익명 데이터

앱 개선을 위해 수집할 수 있는 익명화된 데이터에 대해 알려드립니다

앱 개선을 위해 다음 익명화된 데이터를 수집할 수 있습니다:

앱 사용 통계

어떤 기능이 자주 사용되는지, 화면 조회 횟수 등

기기 정보

기기 유형, 운영체제 버전 (개인 식별 불가)

오류 보고서

앱 개선을 위한 기술적 오류 정보

성능 데이터

기능별 사용 빈도, 성능 데이터

Google Analytics 사용

Google Firebase Analytics를 통해 앱 성능 및 사용자 경험 개선을 위한 익명 데이터를 수집합니다. 기기 설정에서 분석 데이터 수집을 선택적으로 거부할 수 있습니다.

로컬 데이터 저장

앱 기능을 위해 기기에만 저장되는 데이터에 대해 알려드립니다

앱 기능을 위해 다음 데이터를 기기에만 저장합니다:

앱 환경설정

선택한 기준 통화, 언어 설정, 테마 설정

환율 캐시

오프라인 기능을 위해 임시로 캐시된 환율

통화 이력

빠른 접근을 위한 최근 사용한 통화 쌍

앱 설정

표시 설정 및 알림 설정

기기 전용 저장

모든 데이터는 기기의 보안 저장소에 안전하게 보관됩니다. 데이터는 기기를 벗어나지 않으며, 외부 서비스와 동기화되지 않습니다. 앱 삭제 시 저장된 모든 데이터가 영구적으로 제거됩니다.

여러분의 권리

개인정보와 관련된 여러분의 권리를 알려드립니다

Exchango 사용자는 다음 권리를 갖습니다:

분석 거부 권리

기기 설정에서 분석 데이터 수집을 거부할 수 있습니다

데이터 삭제 권리

앱 삭제 시 모든 로컬 데이터가 완전히 제거됩니다

정보 요청 권리

데이터 처리에 대한 정보를 언제든 요청할 수 있습니다

투명성 권리

모든 데이터 처리에 대해 명확히 알 권리가 있습니다

절대 판매하지 않아요

개인정보는 제3자에게 판매하지 않으며, 광고 목적으로 사용하지 않습니다. 수집되는 모든 분석 데이터는 오직 앱 개선 목적으로만 사용됩니다.
