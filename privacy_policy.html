개인정보처리방침

시행일 · 2024년 1월 1일 · 최종 수정일 · 2026년 4월 29일

Exchango AI(이하 "회사")는 이용자의 개인정보를 소중히 여기며, 개인정보보호법 및 관련 법령을 준수합니다. 본 처리방침은 회사가 어떤 정보를, 왜, 어떻게 처리하는지를 명확히 설명합니다.

한 줄 요약

가입이 필요 없습니다. 계산 내역 · 즐겨찾기 · 설정은 모두 여러분의 기기 안에만 저장되며, 회사는 이용자를 식별할 수 있는 어떤 정보도 수집하지 않습니다.

1. 수집하는 정보

1.1 회사가 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 계정 정보 — 로그인이나 회원가입이 없습니다
  • 계산 내역, 즐겨찾기, 정산·팁 기록 — 모두 기기 내부 저장
  • 연락처, 사진, 주소록 등 민감 정보

1.2 자동으로 수집되는 익명 정보

  • 환율 요청 정보 — 통화 코드와 요청 시각. 이용자를 특정할 수 있는 정보는 포함되지 않습니다.
  • 익명 사용 통계 — 화면 방문, 충돌 리포트, 앱 버전, OS 버전, 언어 설정. 모두 익명 처리되며 개인 식별 정보와 결합되지 않습니다.

1.3 기기 내부에만 저장되는 정보

다음 데이터는 이용자의 기기 안에만 저장되며, 외부로 전송되지 않습니다.

  • 환율 계산 · 정산 · 팁 계산 히스토리
  • 즐겨찾기 통화, 기준 통화, 카드 수수료 설정
  • 언어, 테마, 글꼴 크기 등 사용자 설정

2. 정보 이용 목적

  • 실시간 환율 제공 및 서비스 운영
  • 오프라인 상황에서도 마지막 환율로 계산이 가능하도록 캐싱
  • 앱 안정성 개선, 충돌 분석, 버그 수정
  • 기능 사용 빈도 분석을 통한 사용자 경험 개선
  • 법령에 따른 의무 이행

3. 외부 서비스

회사는 이용자의 정보를 제3자에게 판매하거나 마케팅 목적으로 제공하지 않습니다. 다만 서비스 제공을 위해 다음 범주의 외부 서비스를 이용하며, 모두 익명 처리된 데이터만 전달됩니다.

  • 환율 데이터 공급 — 신뢰할 수 있는 환율 정보 제공처로부터 시세 데이터를 받습니다.
  • 익명 분석 서비스 — 앱 안정성과 사용성 개선을 위한 익명 통계 수집.

4. Ango AI · 자연어 비서

Ango AI는 이용자의 기기 내부에서 직접 동작하는 온디바이스(on-device) 모델입니다. 입력하신 질문과 대화는 외부 AI 서비스로 전송되지 않으며, 회사 서버에서도 처리되지 않습니다.

  • 대화 처리는 100% 기기 내부에서 실행
  • 대화 내용 외부 전송 없음
  • 대화 기록은 기기 안에만 저장되며, 언제든 삭제 가능

5. 권한

앱이 요청하는 OS 권한과 그 사용 목적은 다음과 같습니다. 모든 권한은 OS 설정에서 언제든지 회수할 수 있습니다.

  • 위치 (선택) — 현지 통화 자동 추천을 위해 사용. 백그라운드에서 사용하지 않습니다.
  • 인터넷 — 실시간 환율 데이터 수신.
  • 알림 진동 · 부팅 시 자동 시작 — 홈 화면 위젯의 환율 자동 업데이트.

6. 정보 보관 및 파기

회사는 수집한 정보를 목적 달성에 필요한 최소 기간만 보관합니다.

  • 환율 요청 정보: 30일 후 자동 삭제
  • 충돌 리포트: 90일 후 자동 삭제
  • 익명 사용 통계: 분석 서비스의 기본 보관 정책에 따름
  • 기기 내부 데이터(히스토리·설정): 사용자가 직접 삭제하거나 앱을 제거하면 즉시 삭제

7. 데이터 보안

  • 모든 외부 통신은 암호화(HTTPS)로 보호됩니다.
  • 기기 내부 데이터는 운영체제가 제공하는 안전한 저장소에 저장됩니다.
  • 앱을 삭제하면 모든 로컬 데이터가 함께 제거됩니다.

8. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 설정에서 모든 히스토리 · 즐겨찾기 일괄 삭제
  • Ango AI 대화 기록 즉시 삭제
  • 익명 분석 옵트아웃 (설정 → 개인정보)
  • 앱 삭제 시 기기 내부 데이터 전부 자동 삭제 — 별도 탈퇴 절차 없음
  • 위치 등 OS 권한은 설정에서 언제든 회수 가능

9. 정책 변경

본 방침은 법령이나 서비스의 변경사항을 반영하기 위해 수정될 수 있습니다. 중요한 변경이 있을 경우 앱 내 공지 및 본 페이지 최상단의 시행일·수정일로 안내하며, 시행일로부터 효력이 발생합니다.
