Exchango 서비스 이용을 위한 개인정보 처리 정책

개인정보처리방침 안내

Exchango는 이용자의 개인정보를 중요시하며, 개인정보보호법 및 관련 법령을 준수합니다. 본 개인정보처리방침은 Exchango가 제공하는 환율 계산기 앱 서비스(이하 "서비스")와 관련하여 이용자의 개인정보가 어떻게 수집, 이용, 보호되는지를 설명합니다.

✓ 시행일: 2024년 1월 1일
✓ 최종 수정일: 2024년 12월 1일
✓ 문의: exchango.app@gmail.com
2

수집하는 정보

서비스 제공을 위해 최소한의 정보만을 수집합니다

기기 정보 (버그 제보 시)

Exchango는 일상적인 사용 중에는 기기 정보를 수집하지 않습니다. 단, 사용자가 버그를 제보할 때 문제 해결을 위해 기기 유형, 운영체제 버전, 앱 버전 등의 기술적 정보가 포함될 수 있습니다.

✓ 버그 제보 시에만 기기 정보 제공
✓ 자동 수집이 아닌 사용자 제공 방식
✓ 문제 해결 목적으로만 사용

사용자 설정

개인화된 서비스 제공을 위해 사용자가 직접 설정한 정보를 기기 내부에 저장합니다. 이 정보는 서버로 전송되지 않으며, 오직 사용자의 기기에만 보관됩니다.

✓ 선호 언어 설정
✓ 기준 통화 설정
✓ 즐겨찾기 통화 목록

사용 기록

환율 계산, 팁 계산, 정산 기록 등의 사용 내역을 기기에 저장하여 편리한 재사용을 가능하게 합니다. 모든 기록은 로컬에만 저장되며 외부로 전송되지 않습니다.

✓ 환율 계산 히스토리
✓ 팁 계산 기록
✓ 정산 내역

정보 사용 방법

수집된 정보는 서비스 제공과 개선에만 사용됩니다

정보 활용 목적

Exchango는 수집한 정보를 다음과 같은 목적으로만 사용합니다:

✓ 실시간 환율 정보 제공 및 정확한 환율 계산
✓ 오프라인 모드에서의 서비스 제공
✓ 사용자 맞춤형 인터페이스 제공
✓ 앱 성능 최적화 및 버그 수정
✓ 서비스 품질 개선 및 새로운 기능 개발
4

데이터 저장 및 보안

업계 표준 보안 조치로 정보를 안전하게 보호합니다

로컬 저장

모든 개인 설정과 사용 기록은 사용자의 기기에만 저장됩니다. 외부 서버로 전송되지 않으며, 앱을 삭제하면 모든 데이터가 완전히 제거됩니다.

✓ 기기 내부 안전한 저장소 사용
✓ 외부 서버 전송 없음
✓ 앱 삭제 시 완전 삭제

보안 조치

API 통신 시 암호화된 연결(HTTPS)을 사용하며, 시간 기반 토큰 인증으로 무단 접근을 방지합니다. 모든 통신은 보안 프로토콜을 준수합니다.

✓ HTTPS 암호화 통신
✓ 시간 기반 토큰 인증
✓ CDN을 통한 안전한 데이터 전송
5

제3자 서비스

환율 데이터 제공을 위한 최소한의 외부 서비스만 사용합니다

외부 서비스 사용

Exchango는 정확한 환율 정보 제공을 위해 신뢰할 수 있는 외부 서비스를 통해 환율 데이터를 제공받습니다:

✓ 실시간 환율 데이터 API 활용
✓ 보안 프로토콜을 통한 안전한 데이터 전송
⚠️ 개인정보는 제3자와 공유되지 않습니다
⚠️ 오직 환율 데이터 요청만 전송됩니다
6

이용자의 권리

귀하는 언제든지 자신의 정보를 관리할 수 있습니다

정보 열람권

앱 설정에서 저장된 모든 개인 설정과 사용 기록을 확인할 수 있습니다. 모든 데이터는 투명하게 표시되며 언제든지 확인 가능합니다.

정보 삭제권

앱 설정에서 계산 기록, 즐겨찾기, 사용 내역을 선택적으로 삭제하거나 앱을 삭제하여 모든 데이터를 완전히 제거할 수 있습니다.

서비스 중단권

언제든지 앱 사용을 중단할 수 있으며, 앱 삭제 시 모든 로컬 데이터가 자동으로 삭제됩니다. 별도의 탈퇴 절차가 필요하지 않습니다.

문의 및 신고

개인정보 관련 문의사항이나 우려사항이 있으시면 언제든지 exchango.app@gmail.com으로 연락주시기 바랍니다.

아동의 개인정보 보호

만 14세 미만 아동의 개인정보를 수집하지 않습니다

아동 보호 정책

Exchango는 모든 연령의 사용자가 안전하게 사용할 수 있도록 설계되었습니다. 앱에서 생성되는 모든 데이터(계산 기록, 설정 등)는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다. 만 14세 미만 아동이 사용하더라도 개인식별정보는 수집되지 않으며, 모든 데이터는 로컬에만 보관됩니다. 부모님께서는 안심하고 자녀가 앱을 사용하도록 허용하실 수 있습니다.
