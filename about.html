Main Features

Provides various currency calculations and convenience features

1. Currency Calculator

An intuitive smart currency calculator that anyone can easily use. Supports over 150 currencies and provides accurate calculation results using exchange rate information. Convert any amount to another currency instantly with just a few simple touches.

✓ Automatic exchange rate data updates
✓ Bidirectional calculation and swap functionality
✓ Precise decimal point calculations

2. Settlement Function

Resolve complex multi-currency settlements from overseas travel or international business at once. Automatically calculates amounts paid by multiple people in different currencies to provide fair settlement results.

✓ Fair cost sharing calculation per participant
✓ Multi-currency mixed settlement support

3. Tip Calculator

Calculate tips easily and accurately for restaurants, taxis, and service industries. Reflects various countries' tip cultures and ratios to calculate appropriate tips, and also supports split calculations according to the number of people.

✓ Regional tip culture reflection
✓ Tip split calculation per person
✓ Tip calculation history feature

4. Today's Exchange Rates

Get real-time exchange rate information and fluctuation status of major currencies worldwide at a glance. Exchange rate data is presented through visual charts for intuitive checking, and frequently checked currencies can be registered as favorites for convenient monitoring.

✓ Exchange rate status chart provided
✓ Daily volatility and trend display
✓ Favorites function management

5. Multi-Currency Calculator

A powerful feature that allows you to compare multiple currencies simultaneously with a single input. For example, you can see at once how much 1000 USD is in Korean Won, Japanese Yen, and Euro on one screen, helping you choose the optimal currency.

✓ Multiple currency simultaneous calculation
✓ Currency comparison analysis

6. Games (Ladder & Roulette)

Fun game features that let you make various decisions with friends. Solve everyday choice dilemmas like choosing travel destinations, deciding lunch menus, or team divisions through exciting games. Provides both ladder games and roulette games.

✓ Classic ladder game
✓ Colorful roulette game
✓ Custom item settings

7. Various Settings

Customize the app according to your personal usage patterns and environment. Various options including base currency, language, theme, and font size provide an optimal user experience.

✓ Base currency settings
✓ Language settings (7 languages support)
✓ Theme settings (Light/Dark)
✓ Font size settings
